Tentang Halaman Ini

Cara Kerja Cache

01 —

Cache MISS

Request pertama ke /page.html selalu MISS karena belum tersimpan. Proxy meneruskan ke webserver.

02 —

Cache HIT

Request berikutnya akan langsung dilayani dari cache proxy tanpa menghubungi webserver lagi. Lebih cepat.

03 —

Multithreading

Proxy dan webserver membuat thread baru untuk setiap koneksi masuk, sehingga banyak client bisa dilayani bersamaan.
